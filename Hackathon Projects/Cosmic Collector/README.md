# Cosmic Collector

Cosmic Collector is a four-person Robotech 2026 hackathon project focused on autonomous cave navigation and ore marking in a Mars-inspired environment.

## Links

- **Original Repository:** https://github.com/Hiptostee/robotech
- **Devpost:** https://devpost.com/software/cosmic-collector

## Project Summary

Cosmic Collector is a mecanum-drive robot built to explore cave-like environments, avoid walls, identify rocks with onboard vision, and drop physical markers for later retrieval. The system combines a Raspberry Pi, camera, LiDAR, IMU, custom ESP32 firmware, and ROS-based reactive navigation.

On the software side, the project centered on:

- reactive cave navigation using LiDAR data
- ROS 2 integration for sensing and control
- custom ESP32 firmware for low-level hardware control
- sensor filtering and watchdog logic for safer real-time behavior
